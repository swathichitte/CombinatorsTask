COMBINATORS

  • A combinator is something that explains the relationship between the selectors
  • A CSS selector can contain more than one simple selector
  • Between the simple selectors, we can include a combinator

TYPES OF COMBINATORS

  1. Descendant combinator (space)
  2. Child combinator (>)
  3. Next sibling combinator (+)
  4. Subsequent-sibling combinator (~)

1.Descendant combinator

It is represented by space( ) and it will selects the all elements of any level within the specifed element.

Example:-

< style >
div p {
background-color: yellow;
}
< /style >

< div >
< p >Paragraph 1 in the div.< /p >
< p >Paragraph 2 in the div.< /p >
< section >< p >Paragraph 3 in the div.< /p >< /section >
< /div >

< p >Paragraph 4. Not in a div.< /p >
< p >Paragraph 5. Not in a div.< /p >

2.Child combinator

It is represented by (>) and it will selects only direct children of the specifed element.

Example:-

< style >
div > p {
background-color: yellow;
}
< /style >
< div > < p >Paragraph 1 in the div.< /p >
< p >Paragraph 2 in the div.< /p >
< section >
< p >Paragraph 3 in the div (inside a section element).< /p >
< /section >

Paragraph 4 in the div.< /p >
< /div >

< p >Paragraph 5. Not in a div.< /p >
< p >Paragraph 6. Not in a div.< /p >

3.Next sibling combinator

The next sibling combinator is used to select an element that is directly after another specific element.

Sibling elements must have the same parent element, and "adjacent" means "immediately following".

Example:-

< style >
div+p {
background-color: yellow;
}
< /style >
< div > < p >Paragraph 1 in the div.< /p >
< p >Paragraph 2 in the div.< /p >
< section >
< p >Paragraph 3 in the div (inside a section element).< /p >
< /section >

Paragraph 4 in the div.< /p >
< /div >

< p >Paragraph 5. Not in a div.< /p >
< p >Paragraph 6. Not in a div.< /p >

4.subsequent-sibling combinator

The subsequent-sibling combinator selects all elements that are next siblings of a specified element.

Example:-

< style >
div~p {
background-color: yellow;
}
< /style >
< div > < p >Paragraph 1 in the div.< /p >
< p >Paragraph 2 in the div.< /p >
< section >
< p >Paragraph 3 in the div (inside a section element).< /p >
< /section >

Paragraph 4 in the div.< /p >
< /div >

< p >Paragraph 5. Not in a div.< /p >
< p >Paragraph 6. Not in a div.< /p >
